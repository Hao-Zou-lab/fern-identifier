# Fern Species Identifier

**Powered by CLIP (Fine-tuned)**

---

## What is this?

A simple tool to identify two similar fern species:
- **Leptochilus kepingii**
- **Leptochilus digitatus**

---

## How to use

1. Double-click `fern_identifier.exe`
2. Click "Select Image" button
3. Choose a fern photo (JPG, PNG, JPEG, BMP, TIFF)
4. View the identification result

---

## System Requirements

- Windows 10 or Windows 11
- No Python installation required

## Notes

- The two species are very similar in appearance
- Best results with clear, well-lit photos
- Close-up of leaves or veins gives better accuracy

---

## Troubleshooting

If the program doesn't open:
- Make sure `ensemble_weights.pt` is in the same folder as the `.exe`
- Try running as administrator
